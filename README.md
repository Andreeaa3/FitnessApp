FitnessApp

A mobile application built with React Native and Expo designed to help users track their fitness journeys, log workouts, and follow structured meal plans.

Features
- Daily Dashboard: Allows users to view their daily step progress and logged workouts.
- Step Counter: Tracks daily steps, allows adding steps in increments of 100, and saves history with timestamps.
- Workout History: Allows users to log new activities, view previous exercises, and manage their workout logs.
- Stopwatch: Features a built-in timer to track exercise durations and view a list of previous time sessions.
- BMI Calculator: Calculates the Body Mass Index based on user input for weight in kilograms and height in centimeters.
- Workout Ideas: Displays a curated list of exercises including target muscle groups, required equipment, and detailed instructions.
- Custom Meal Plans: Provides structured nutritional guides and recipes tailored for different goals, such as Weight Loss, Muscle Gain, Energy and Performance, and Vegetarian or Vegan diets.
